# Main subject

Наше приложение призвано помочь пользователю максимально грамотно и эффективно управлять своим временем. Мы - не просто таск трекер для повседневных задач, мы - интеллектуальная система, проводящая пользователя через самые эффективные методики планирования времени. У нас есть весь базовый функционал похожих приложений: задачи, привычки, проекты, аналитика. Разве что мы стараемся сделать это максимально интуитивно для людей. Чтобы даже человек, никогда не сталкивавшийся с техниками тайм-менеджмента, смог заметно увеличить свою продуктивность, пользуясь нашим приложением.
